# Saving

一款帮你在[资料网](https://www.zip118.com/)省钱的命令行下载工具。

## 如何使用

1. 打开命令行下载安装：`git clone git@github.com:LinusLing/saving.git && cd saving && npm i`
2. 查询关键入参并执行：`node index.js [cid] [aid] [directory_name]`
3. 完成后，资料就在当前目录的 `downloads` 文件夹下了

> 自行保证 node / npm 的可用性
